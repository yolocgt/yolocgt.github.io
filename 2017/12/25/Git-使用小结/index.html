Git 使用小结

git安装完成后,还需要最后一步设置,在命令行输入:

git config –global user.name ‘yourname’
git config –global user.email ‘yourqqnum@qq.com’

1. 初始化一个Git仓库，使用 git init 命令。

2. 添加文件到Git仓库，分两步：

3. 要随时掌握工作区的状态，使用 git status 命令。

4. HEAD指向的版本就是当前版本，因此，Git允许我们在版本的历史之间穿梭，使用命令git reset –hard commit_id。

5. 修改后可用命令来查看工作区和版本库文件的区别:

git diff readme.txt

6. 远程仓库

创建SSH Key :

ssh-keygen -t rsa -C ‘yourqqnum@qq.com’
查看public key,并把它添加到码云的 SSH公钥 :
cat ~/.ssh/id_rsa.pub

7.要关联一个远程库，使用命令git remote add origin git@server-name:path/repo-name.git；
